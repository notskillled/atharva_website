Hackathon – The Mosquito Guard

Description: A low-cost, battery-powered ESP32 sensor system that monitors environmental conditions temperature, humidity made for communities heavily impacted by mosquito-borne diseases.

Cause

  • Mosquitoes cause at least 2.7 million deaths every year
  • About 500 million cases of mosquito-borne diseases occur annually
  • 90% of mosquito-borne illnesses occur in Africa

Features

  • $30 Build that targets real time mosquito habitat detection.
  • Accurately detects areas favorable for mosquito breeding
  • Python-based algorithms

Tech Stack

  • Python
  • ESP32 + Sensors: DHT22/BME280 for temp & humidity
  • Website + Data Analysis

Impact

  • Mosquitoes are more than just a nuisance—they pose serious health risks by spreading diseases such as malaria, dengue, and Zika. Identifying areas with high mosquito activity is crucial to preventing bites, reducing disease transmission, and protecting communities. Our Mosquito Guard system detects and maps mosquito hotspots, enabling targeted interventions and smarter prevention strategies.

Autonomous Racing Drone Controller

A motion controller designed for high-speed autonomous racing drones with automatic PID calibration.

Problem

Manual PID tuning was slow and inconsistent, making rapid testing and iteration difficult during competitions.

Design & Planning

  • Researched existing PID tuning methods
  • Designed calibration routine for STM32
  • Defined performance metrics (stability & response time)
Initial PID control model

Build & Implementation

Implemented the control loop in Python with real-time sensor feedback.

Control loop implementation
Drone hardware integration

Testing & Iteration

  • Bench testing for stability
  • Flight tests with logged telemetry
  • Iterated PID gains based on response curves

Results & Takeaways

Reduced tuning time by ~60% and achieved smoother autonomous flight. Learned how control theory translates to real-world hardware.
